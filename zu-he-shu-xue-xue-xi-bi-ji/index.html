Prufer 序列

Prufer 序列是一种树形结构和序列相互映射的规则。（例如 DFS 序。）

与其他序列的区别

DFS 序：将一棵子树映射为一段连续的区间。

欧拉序：可应用为二叉搜索树。

Prufer 序列是和树的双向唯一映射。这意味着可以有树得到唯一的 Prufer 序列，反之也可以唯一的得到一棵树。

同时，Prufer 序列也包含了所有节点的度数和连接关系。

使用场景

可以将构造树转化为构造序列，将统计树的信息转化为统计序列的信息，将树上 DP 转化为序列 DP。

如何得到 Prufer 序列

1.统计树上所有结点的度数 degideg_idegi​；

2.找到所有度数为 111 的节点中编号最小的结点 xxx。

3.令 Prufer 序列 pi=faxp_i=fa_xpi​=fax​，将 degfax←degfax−1deg_{fa_x}\gets deg_{fa_x} - 1degfax​​←degfax​​−1。

4.重复步骤 2-3，直到剩余两个点时结束。

Prufer 序列的性质

1.结点 xxx 在 Prufer 序列中出现的次数 yyy 满足 y+1=degxy+1=deg_xy+1=degx​。

2.编号最大点 nnn 一定是剩下的 222 个结点之一。

void to_prufer(){//一般来说，在构造 prufer 序列的时候，我们会让结点 n 作为根节点。
  int x;

    }
  }

fact[0] = 1;
for (int i = 1; i <= n; i++){
  fact[i] = fact[i - 1] * i;
}

void decantor(int n, int x){//长度为 n 的排列的排名为 x
  x--;
  vector<int> tmp(n);//存储当前未出现在排列中的数字
  vector<int> a(n);//排列
  for (int i = 0; i < n; i++){
    tmp[i] = i + 1;
  }
  for (int i = 1; i <= n; i++){
    int p = x / fact[n - i];
    x %= fact[n - i];
    a[i - 1] = tmp[p];
    tmp.erase(lower_bound(tmp.begin(), tmp.end(), a[i - 1]));
  }
}
